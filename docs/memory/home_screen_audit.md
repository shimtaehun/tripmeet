# 홈 화면 감사 보고서 (2026-03-03)

## 현재 구성 (문제 없음)
- 히어로 이미지 + 검색바
- 빠른 메뉴 (5개: AI 일정, 매칭, 커뮤니티, 맛집, 동행)
- AI 일정 배너
- 인기 여행지 카드 (6개)
- GPS 안전 안내 텍스트

## 발견된 문제 (7가지)

### [P1] 사용자 인사말 없음
- 로그인한 유저의 이름/닉네임을 전혀 활용하지 않음
- "어디로 떠나세요?" 만 노출, 개인화 제로
- 수정: Supabase auth에서 user_metadata.full_name 가져와 "OOO님, 어디로?" 표시

### [P1] 콘텐츠 섹션 심각하게 부족
- 홈 화면에서 앱의 핵심 기능 3개(커뮤니티, 동행, 여행자)를 전혀 미리 보여주지 않음
- 빠른 메뉴 아이콘만 있고 실제 콘텐츠 미리보기가 없어 진입 장벽 높음

### [P1] 근처 여행자 미리보기 없음
- 메인 기능인 "여행자 매칭"의 실제 여행자 카드를 홈에서 보여줘야 함
- 매칭 탭 진입 동기 부여 부족

### [P1] 커뮤니티 최신 글 없음
- 커뮤니티가 앱 핵심 기능인데 홈에서 글 한 줄도 미리 보여주지 않음
- 커뮤니티 탭 진입 동기 부여 부족

### [P2] 동행 구인 HOT 게시글 없음
- "다음 달 도쿄 동행 구해요" 같은 카드가 홈에 있으면 즉시 관심 유발

### [P2] 통계/신뢰 지표 없음
- "현재 1,247명 여행 중" 같은 수치가 히어로에 없음
- 랜딩 페이지엔 있는데 실제 앱 홈에는 없는 모순

### [P3] 섹션이 모두 단색 배경으로 단조로움
- 빠른 메뉴, AI 배너, 여행지 카드가 시각적으로 구분되지 않고 이어짐
- 섹션 간 배경색 변화, 구분선 필요

## 수정 계획
1. 히어로에 사용자 이름 + 통계 배지 추가
2. 근처 여행자 미리보기 섹션 추가 (2명 카드)
3. 커뮤니티 최신 글 미리보기 섹션 추가 (2개 카드)
4. 동행 구인 HOT 섹션 추가 (2개 카드)
5. 섹션별 배경색 교차 적용으로 시각적 리듬감 부여
